# 마이링크 (MyLink) 🔗

마이링크는 나만의 모든 활동과 콘텐츠를 단 하나의 링크로 연결하는 **링크트리(Linktree) 클론 서비스**입니다.

이 프로젝트는 Next.js와 shadcn/ui를 기반으로 구축된 템플릿을 사용합니다.

## 📌 주요 문서 바로가기
* **[기능 정의서 (PRD)](./docs/prd.md)**: 서비스의 프로젝트 개요, 핵심 기능 및 요구사항 상세 내역을 확인할 수 있습니다.

## 🛠️ 프로젝트 개요
* **목적**: 다중 링크 연결 및 개인 브랜딩을 돕는 마이크로 랜딩 페이지 플랫폼 구축
* **대상 사용자**: SNS 크리에이터, 1인 창업자, 브랜드 마케터, 포트폴리오 관리가 필요한 구직자 등
* **핵심 기술 스택**: Next.js, TypeScript, Tailwind CSS, shadcn/ui

## 🚀 개발 가이드

### 컴포넌트 추가하기
앱에 새로운 컴포넌트를 추가하려면 다음 명령어를 실행하세요:

```bash
npx shadcn@latest add [component-name]
```

UI 컴포넌트는 `components/ui` 디렉토리에 생성됩니다.

### 컴포넌트 사용하기
추가한 컴포넌트를 다음과 같이 가져와서 사용할 수 있습니다:

```tsx
import { Button } from "@/components/ui/button";
```
