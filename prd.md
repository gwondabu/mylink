# [PRD] 링크트리 클론 서비스 "마이링크 (MyLink)" 기능 정의서

본 문서는 멀티 링크 페이지 제작 서비스인 **마이링크(MyLink)**의 제품 요구사항 정의서(PRD)입니다. 사용자가 하나의 링크를 통해 자신의 다양한 SNS, 포트폴리오, 비즈니스 채널을 효과적으로 홍보할 수 있는 서비스를 정의합니다.

---

## 1. 프로젝트 개요

### 1.1 프로젝트명
* **마이링크 (MyLink)**

### 1.2 목적
* 인스타그램, 틱톡, 트위터 등 주요 SNS 플랫폼의 '프로필 내 단 하나의 링크만 허용'하는 제약을 극복합니다.
* 사용자가 개성 있는 개인 프로필 페이지를 커스텀하여 자신의 다중 링크와 소셜 미디어 정보를 한눈에 보여줄 수 있는 마이크로 랜딩 페이지 서비스를 제공합니다.
* 심플하고 직관적인 UI/UX를 제공하여 기술적 지식이 없는 일반 사용자도 3분 안에 프로필 페이지를 개설할 수 있도록 돕습니다.

### 1.3 대상 사용자
* **SNS 크리에이터 및 인플루언서**: 유튜브, 인스타그램, 블로그 등 여러 플랫폼의 채널을 동시에 홍보하고자 하는 사용자.
* **1인 창업자 및 브랜드 마케터**: 자사 쇼핑몰, 이벤트 페이지, 카카오톡 채널 등을 한곳에서 모아 잠재 고객에게 연결하려는 사용자.
* **프리랜서 및 구직자**: 포트폴리오, 링크드인, 경력 기술서 등을 시각적으로 정리하여 전달하고자 하는 사용자.
* **일반 사용자**: 자신을 나타내는 여러 링크(오픈채팅, 노션 페이지 등)를 하나로 묶어 공유하고 싶은 사람.

## 3. 기능별 상세 설명

### 3.1 필수 기능 (Must-Have)

#### 3.1.1 이메일 회원가입 및 로그인
* **화면 정의**: 회원가입 화면, 로그인 화면
* **상세 사양**:
  * 아이디(이메일), 비밀번호, 비밀번호 확인을 입력받아 가입 처리를 진행합니다.
  * 비밀번호는 단방향 해시 암호화(SHA-256 또는 bcrypt)를 통해 안전하게 보관합니다.
  * 로그인 성공 시 세션 혹은 JWT 토큰을 발행하여 사용자 인증을 유지합니다.

#### 3.1.2 사용자 고유 주소 설정
* **상세 사양**:
  * 회원가입 시점에 중복되지 않는 영문/숫자 기준의 `username`을 입력받습니다.
  * 사용자의 개별 프로필 랜딩 페이지는 `https://mylink.to/{username}` 구조로 매핑됩니다.
  * 이미 등록된 고유 주소는 타인이 선점할 수 없도록 실시간 중복 체크를 지원합니다.

#### 3.1.3 링크 관리 (CRUD)
* **화면 정의**: 관리자 대시보드 내 링크 편집 탭
* **상세 사양**:
  * **추가(C)**: '새 링크 추가' 버튼을 눌러 링크 카드를 생성하고 제목(Title)과 연결 주소(URL)를 입력합니다.
  * **조회(R)**: 사용자가 등록한 링크 카드 목록을 대시보드 화면에서 확인할 수 있습니다.
  * **수정(U)**: 활성화 상태(토글 버튼으로 On/Off), 텍스트 타이틀, URL 주소를 언제든 수정할 수 있습니다.
  * **삭제(D)**: 불필요한 링크 카드는 휴지통 아이콘을 클릭하여 즉시 삭제할 수 있습니다.

#### 3.1.4 드래그 앤 드롭을 통한 링크 순서 변경
* **상세 사양**:
  * 각 링크 카드의 좌측에 드래그 핸들러 아이콘을 배치합니다.
  * 마우스 드래그 또는 터치 드래그를 통해 링크 카드들의 수직 순서를 자유롭게 바꿀 수 있습니다.
  * 순서 변경 시 데이터베이스 혹은 내부 상태 값에 정렬 순서(Order Index)가 자동으로 갱신됩니다.

#### 3.1.5 프로필 설정 커스텀
* **상세 사양**:
  * **프로필 이미지**: 사용자가 직접 원형 영역에 들어갈 이미지를 업로드할 수 있으며, 미설정 시 기본 아바타 이미지를 노출합니다.
  * **프로필 이름**: 페이지 상단에 굵은 글씨로 노출될 이름 혹은 닉네임을 설정합니다. (공백 포함 최대 20자)
  * **한 줄 소개**: 프로필 이름 하단에 들어갈 짧은 자기소개 글을 설정합니다. (최대 80자)

#### 3.1.6 디자인 및 테마 설정
* **상세 사양**:
  * 다양한 사전 정의 테마(예: Classic Light, Premium Dark, Pastel Sky, Sunset Glow 등)를 제공합니다.
  * 테마 선택 시 배경색(혹은 그라디언트 배경), 버튼 라운드 크기(둥근 사각형, 직사각형, 타원형), 글자 색상이 프리셋에 맞춰 한 번에 변경됩니다.
  * 대시보드 우측 또는 별도 영역에 '실시간 모바일 미리보기(Live Preview)' 컴포넌트를 배치하여, 설정한 디자인을 저장하기 전에 모바일 화면 형태로 실시간 확인이 가능하도록 합니다.

#### 3.1.7 반응형 모바일 뷰 페이지 (랜딩 페이지)
* **상세 사양**:
  * 실제 외부 사용자들이 접속하는 화면으로, 모바일 세로 화면 비율에 가장 최적화된 레이아웃을 갖습니다.
  * 프로필 이미지, 닉네임, 한 줄 소개가 상단 중앙에 정렬되어 배치되며, 그 아래로 등록된 활성 링크들이 버튼 카드의 형태로 순서대로 정렬됩니다.
  * 클릭 시 대상 URL이 새 창(`_blank`)으로 열리도록 설계하여 사용자가 기존 마이링크 페이지를 이탈하지 않도록 유도합니다.

### 3.2 선택 기능 (Nice-to-Have)

#### 3.2.1 소셜 간편 로그인
* **상세 사양**:
  * OAuth 2.0 프로토콜을 활용하여 구글, 카카오 등의 계정으로 가입 및 로그인이 가능하게 합니다.
  * 복잡한 비밀번호 입력 없이 첫 진입 허들을 낮춥니다.

#### 3.2.2 소셜 아이콘 연동
* **상세 사양**:
  * 인스타그램, 페이스북, 유튜브, X(트위터), 틱톡, 이메일, 깃허브 등 주요 플랫폼 아이콘의 활성 링크를 설정할 수 있습니다.
  * 활성화된 소셜 미디어 아이콘들은 텍스트 링크 카드 위 혹은 아래에 일렬로 깔끔하게 배치됩니다.

#### 3.2.3 방문 및 클릭 통계 분석
* **상세 사양**:
  * 사용자의 고유 마이링크 페이지 전체 방문 횟수(PV) 및 고유 방문자수(UV)를 집계합니다.
  * 각 텍스트 링크 버튼이 클릭된 횟수와 클릭률(CTR) 정보를 차트나 수치 카드로 대시보드에 제공합니다.
  * (심화) 유입 브라우저(모바일/데스크톱), 유입 국가 정보를 통계 화면에서 파악할 수 있도록 지원합니다.

#### 3.2.4 SEO 및 SNS 공유 설정
* **상세 사양**:
  * 검색 엔진 최적화(SEO)를 위해 페이지 메타 타이틀(Meta Title)과 메타 설명(Meta Description)을 사용자가 직접 입력할 수 있는 폼을 구성합니다.
  * 링크 공유 시 노출될 대표 썸네일 이미지(Open Graph Image)를 추가로 등록할 수 있게 합니다.
